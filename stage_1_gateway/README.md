# Stage 1: 智能网关与用户 UI

本模块是整个系统的入口，负责感知用户输入、持久化会话状态并驱动后端任务分发。

---

## 🏗️ 核心功能

### 1. 意图路由 (Intent Router)
*   **语义理解**：利用 LLM 实现从自然语言到任务标签（RAG/ACTION/CHAT）的精准转换。
*   **结构化决策**：通过 Pydantic 约束路由结果，确保后端跳转的确定性。

### 2. 状态机驱动的对话流
*   **LangGraph 集成**：UI 直接挂载编译后的 `graph_agent`。
*   **审批交互**：当后端状态机进入挂起（Approval）节点时，UI 自动渲染审批卡片。
*   **Thread ID 持久化**：为每个用户会话分配唯一的 ID，实现跨页面刷新的状态恢复。

### 3. 多视图 Dashboard
*   **用户对话**：主聊天界面。
*   **监控看板**：实时系统指标。
*   **RAG 实验室**：深度策略对比与 Ragas 自动化评测。
